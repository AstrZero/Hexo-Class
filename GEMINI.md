# Project Instructions: Hexo-Class

## Homework Post Convention
- **Content**: For homework-related posts (usually based on images), do NOT transcribe the text manually unless explicitly requested.
- **Format**: Create a post asset folder and include the image directly in the post.
- **Naming**: Use English slugs for filenames to avoid encoding issues, but keep Chinese titles in the front-matter.
- **Tags**: Always use the tag "作业" for these posts.
